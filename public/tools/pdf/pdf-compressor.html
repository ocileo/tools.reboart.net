FAQ - Pertanyaan yang Sering Diajukan
Berapa ukuran maksimal file PDF?
Maksimal 50MB. Untuk file lebih besar, disarankan kompres terlebih dahulu menggunakan software lain.
Apakah kualitas gambar menurun?
Tergantung tingkat kompresi. Mode "Rendah" menjaga kualitas hampir sama, "Tinggi" mengurangi kualitas untuk ukuran lebih kecil.
Apakah data saya aman?
100% aman! Proses kompresi dilakukan sepenuhnya di browser Anda. File tidak pernah dikirim ke server.
Berapa persen pengurangan ukuran?
Rata-rata 30-70% tergantung konten dan tingkat kompresi yang dipilih.
Apakah ada watermark?
Tidak ada watermark sama sekali. Hasil kompresi bersih dan profesional.
Bisakah kompres PDF yang dilindungi password?
Saat ini tidak mendukung PDF yang dilindungi password. Harap buka proteksi terlebih dahulu.
