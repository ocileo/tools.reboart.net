FAQ - Pertanyaan yang Sering Diajukan
Berapa ukuran maksimal file PDF?
Maksimal 50MB per file. Untuk file lebih besar, disarankan kompres terlebih dahulu.
Apakah data saya aman?
100% aman! Proses split dilakukan sepenuhnya di browser Anda. File tidak pernah dikirim ke server.
Berapa jumlah halaman maksimal?
Tidak ada batasan jumlah halaman. Namun, file dengan banyak halaman mungkin membutuhkan waktu lebih lama.
Apakah ada watermark di hasil?
Tidak ada watermark sama sekali. Hasil split bersih dan profesional.
Bisakah split PDF yang dilindungi password?
Saat ini tidak mendukung PDF yang dilindungi password. Harap buka proteksi terlebih dahulu.
Format halaman apa yang didukung?
Support semua format halaman PDF (A4, Letter, Legal, dll). Format asli akan dipertahankan.
