FAQ - Pertanyaan yang Sering Diajukan
Apakah format hasil konversi bisa diedit?
Ya! Hasil konversi adalah file DOCX yang bisa diedit di Microsoft Word, Google Docs, atau LibreOffice.
Apa perbedaan mode Standard dan OCR?
Standard untuk PDF teks (digital), OCR untuk PDF scan/gambar. OCR lebih lambat tapi lebih akurat untuk scan.
Apakah data saya aman?
100% aman! Proses konversi dilakukan sepenuhnya di browser Anda. File tidak pernah dikirim ke server.
Apakah tabel dan gambar ikut dikonversi?
Ya, aktifkan opsi "Sertakan Gambar" dan "Ekstrak Tabel" untuk hasil yang lebih lengkap.
Berapa batas ukuran file?
Maksimal 50MB per file. Untuk file lebih besar, disarankan di-split terlebih dahulu.
Apakah ada watermark?
Tidak ada watermark sama sekali. Hasil konversi bersih dan profesional.
